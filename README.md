# snakegame
An simple python game application using pygame library

# prequisites
```
python 3.6
pip 
pygame
venv
```
# installation

```
pip install virtualenv
pip install venv
python3 -m venv environment_name
source environment_name/bin/activate
pip install pygame
sudo apt-get update

```
# To run pygame
```
python3 snakegame.py
```
